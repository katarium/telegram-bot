# 🤖 AstroX Telegram Bot

[![Python 3.8+](https://img.shields.io/badge/python-3.8+-blue.svg)](https://www.python.org/downloads/)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)
[![Telegram Bot API](https://img.shields.io/badge/Telegram-Bot%20API-0088cc.svg)](https://core.telegram.org/bots/api)

**AstroX Bot** is a powerful, feature-rich Telegram bot with **25+ commands**, **gradient text formatting**, **interactive buttons**, and a **custom animated banner**. Built with Python and the `python-telegram-bot` library.

## 🌟 Features

✅ **25+ Useful Commands** - Covering fun, utilities, math, system info, and more
✅ **Gradient Text Formatting** - Beautiful colored text in messages
✅ **Interactive Buttons** - Easy navigation with button menus
✅ **Custom Animated Banner** - Eye-catching GIF banner
✅ **Error Handling** - Robust error management with user-friendly messages
✅ **Lightweight** - Minimal resource usage
✅ **Open Source** - Free to use and modify

## 📦 Requirements

- Python **3.8 or higher**
- `pip` (Python package manager)
- Telegram Bot Token ([@BotFather](https://t.me/BotFather))
